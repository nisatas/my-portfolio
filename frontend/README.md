# Nisa Portfolio – Frontend

React + Vite + Tailwind.

## Kurulum

```bash
npm install
```

## Ortam değişkenleri (.env)

- `VITE_API_URL` – Backend API adresi (örn. http://localhost:3001). Tanımlıysa iletişim formu ve NisaBot bu API’yi kullanır.
- `VITE_GEMINI_API_KEY` – (VITE_API_URL yoksa) NisaBot için doğrudan Gemini API key.

## Çalıştırma

```bash
npm run dev
```

Build: `npm run build`
